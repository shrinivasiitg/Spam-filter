To compile: 

1. Go to Matlab command line and type:
parentFile

2.It will ask for the file name to be given as input:
Enter "optdigit.data" or "iris.data" 

3. It will  run the file given as input.

4. Now it will ask for the p value in the polyKernel function. 
Enter the p value. It will run the file polyKernel.m file.

5. It will ask the sigma value for the radial Kernel function.
Enter the sigma value. It will run the radialKernel.m file.
